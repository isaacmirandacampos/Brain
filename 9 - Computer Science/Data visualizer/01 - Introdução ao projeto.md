## O que é o projeto?
O projeto é um visualizador de dados, a ideia é disponibilizar insights através de gráficos interativos.

## Qual a ideia de fazer o projeto?
A ideia de fazer o projeto é treinar diversos conceitos e também provar para o mercado de trabalho as minhas competências. Vou aplicar diversos conceitos como:
- Habilidade de fazer gráficos interativos
- Animações
- SEO
- Habilidade de manipulação de grande quantidade de dados
- Habilidade de trabalhar com ambiente em nuvem
